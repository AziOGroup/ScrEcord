# ScrEcord
  
.NET Framework製デスクトップキャプチャソフトウェアです  
  
◆◇◆使用方法◆◇◆  
・PrintScreenキーまたは通知アイコン左クリックでキャプチャ呼び出し  
・キャプチャを呼び出した後は左上から右下にドラッグすることでキャプチャの範囲を指定することができます  
・通知アイコン右クリックでサブメニューを開きます
